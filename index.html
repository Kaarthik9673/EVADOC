4. Configuration Steps

  1. Search for config.json file in the EVA directory (/home/pi/EVA )
    • Right Click on config.json file and click on View/Edit.
    • Change the tcp IP address as Device IP or required IP
    • Save and close the file
    • In Filezilla Click on YES to reflect the changes in files.

  2. Search for host.json file in the EVA directory (/home/pi/EVA )
    • Right Click on host.json file and click on View/Edit.
    • Change the IP address as Device IP or required IP
    • Save and close the file
    • In Filezilla Click on YES to reflect the changes in files.

  3. After Changing the Config.json and Host.json files, restart the EVA service to apply the changes.

5. Network Setup

  • If Device Connected locally, If the device IP series is 1, the local IP should be in the same series.
  • If Device Connected remotely, ensure proper port forwarding is set up.
  • Use ping to test connectivity.
  • Use telnet to test Communication.
  • Check ifconfig for correct IP assignment.
